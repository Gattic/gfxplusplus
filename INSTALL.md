# Install, Compile, and Run

---

## Dependencies

`cmake`

`make`

`g++`

`libpng-dev`

SDL2 graphics library
`libsdl2-dev`

SDL2 TTF for GUI
`libsdl2-ttf-dev`

SDL2 Image for GUI
`libsdl2-image-dev`

---

## Compilation

```
sh .configure.sh
```
or
```
mkdir build
cd build
cmake ..
make
```

---

## Installation

make install

---

## Uninstall

make uninstall
